Order Service JSON Fix Test

This page tests the fixed JSON structure for the Order Service deliveryAddress field.

Test: Create Order with Object DeliveryAddress

This test sends a POST request to create an order with the new JSON structure where deliveryAddress is an object instead of a string.

Order JSON Structure

The new structure expects deliveryAddress as an object:

{
  "restaurantId": 1,
  "items": [
    {
      "menuItemId": 101,
      "itemName": "Margherita Pizza",
      "quantity": 2,
      "price": 12.99,
      "specialInstructions": "Extra cheese, no olives"
    }
  ],
  "totalAmount": 25.98,
  "deliveryAddress": {
    "streetAddress": "123 Main St",
    "apartmentNumber": "Apt 4B",
    "city": "New York",
    "state": "NY",
    "country": "USA",
    "postalCode": "10001",
    "landmark": "Near Central Park",
    "latitude": 40.7128,
    "longitude": -74.0060
  }
}
